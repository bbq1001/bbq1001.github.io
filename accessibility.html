Accessibility Statement

Our commitment to making MyGymBro accessible to everyone.

Our Commitment

MyGymBro GmbH is committed to ensuring digital accessibility for people with disabilities. We continuously improve the user experience and apply relevant accessibility standards.

We aim to conform to the Web Content Accessibility Guidelines (WCAG) 2.2 Level AA.

Measures Implemented

  • Semantic HTML with correct heading hierarchy
  • Full keyboard navigation support
  • High contrast colors meeting WCAG AA
  • Alternative text for meaningful images
  • Accessible form labels and error associations
  • Visible focus indicators
  • Responsive design with 200% zoom support
  • ARIA labels where additional context is required
  • Skip navigation links for keyboard users

Known Limitations

Some older user-generated content may not fully meet accessibility standards. We are actively working to improve this content.

Testing & Tools

  • WAVE Web Accessibility Evaluation Tool
  • axe DevTools browser extension
  • Manual keyboard navigation testing
  • Screen reader testing (NVDA, VoiceOver)
  • Color contrast analyzers
